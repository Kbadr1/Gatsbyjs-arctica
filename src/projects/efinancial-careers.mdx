###### We worked with eFinancialCareers to enable recruiters worldwide to quickly place job ads. <br/> <br/> eFinancialCareers is the world's leading financial services careers website, the Webstore allows hiring managers and recruiters to quickly purchase job listing credits for multiple regions. <br/> <br/> We developed an Angular JS web application powered by a Java REST API, with a shopping cart to select job credits and take payments for orders.

**Technology**

- HTML5
- CSS3
- AngularJS
- Amazon Web Services
